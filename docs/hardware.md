# 🔧 Hardware Guide — GrowPots ESP32

> This document will be completed in **Week 2** of the development plan.

## Overview

The hardware component of GrowPots is based on the **ESP32** microcontroller, which reads environmental sensors and controls the irrigation relay.

## Components (Planned)

| Component | Description |
|---|---|
| ESP32 Dev Board | Main microcontroller (WiFi + BT) |
| Soil Moisture Sensor | Capacitive or resistive humidity sensor |
| DHT22 / DHT11 | Temperature and air humidity sensor |
| Relay Module | Controls the water pump |
| Water Pump (5V) | Mini submersible pump |
| Power Supply | 5V USB or dedicated PSU |
