# 🚀 Deployment Guide — GrowPots on Raspberry Pi

> This document will be completed in **Week 4** of the development plan.

## Overview

GrowPots is designed to run 24/7 on a **Raspberry Pi** using Docker Compose.

## Requirements (Planned)

- Raspberry Pi 4 (2GB+ RAM recommended)
- Raspberry Pi OS (64-bit, Lite or Desktop)
- Docker & Docker Compose
- Local network with WiFi (same as ESP32)

## Steps (Planned)

1. Install Raspberry Pi OS
2. Install Docker and Docker Compose
3. Clone the repository
4. Configure `.env` with database credentials
5. Run `docker-compose up -d`
6. Access the API at `http://<raspberry-ip>:5050`
