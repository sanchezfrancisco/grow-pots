# GrowPots

A smart system to monitor and manage multiple plant pots. Automates watering, lighting, and soil care. Ideal for urban gardens and IoT plant enthusiasts.

## Features

- Real-time soil & environment monitoring
- Automated watering & lighting control
- Central dashboard for multiple pots
- Web interface
- Open-source & extensible
